# QSkill Internship Project

A modern React-based utility suite featuring:

- 🌍 Text Translator
- 🔐 Random String Generator

## Live Demo

Frontend (Vercel):
[(https://qskill-internship-project.vercel.app/)]

Backend (Render):
https://qskill-internship-project-1.onrender.com

## Features

### Text Translator
- Translate text into multiple languages
- Clean and responsive UI
- Real-time translation using backend API

### Random String Generator
- Generate secure random strings
- Custom string length selection
- Instant generation

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- translate-google

## Installation

### Clone Repository

```bash
git clone https://github.com/MayurTagadkar2309/QSkill-Internship-Project.git
```

### Frontend Setup

```bash
cd qskill-internship
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

## Deployment

- Frontend deployed on Vercel
- Backend deployed on Render
